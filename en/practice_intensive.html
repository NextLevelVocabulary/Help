Intensive Practice

Card are repeating with increasing frequency.

This practice is very similar to Browse only, but this practice is more suitable for learning larger quantities of words, because each flashcard is displayed multiple times during each cycle and the intervals are much shorter.

You will find only two buttons on the screen and the flashcard deck will be rotating only in one direction. Skip button will remove the flashcard from the deck and button Next will display the next card. The fundamental difference is that frequency of each remaining card will increase after the deck completely cycles. The cycle value is displayed in the brackets above the flashcard.

You can setup the starting frequency in menu Preferences. If you choose setting "1-2-3 reps.", this means that first cycle will display each card once, second cycle will display each card twice and all subsequent cycles will display each card trice.

 

Learning will become more intensive for flashcards which you do not know. One card can be displayed up to three times during each cycle.

 

You can tap the card to flip it. Swipe the card to move between the cards. The buttons have same function.
