Flashcard Levels

There are three levels of flashcards, which will learn by default in the following sequence. Note that the Installation Wizard activated some cards for you already, you might see many cards activated as high proficiency Audiocard

English>Czech Flashcard - this card is by default the first learning level and is the ideal card to start learning the word.
Czech>English Flashcard - this card is by default your second learning level. It is typically more difficult to translate to English.
Audio>Czech Flashcard - this card is by default the last learning level. You must be able to understand spoken word in order to fully learn the card. *Some words don't have pronunciation and the audio level will be skipped for these flashcards.
