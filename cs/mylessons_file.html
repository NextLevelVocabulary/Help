Import slovíček ze souboru MS Excel

Pro import slovíček z telefonu musíte mít nainstalovaný nějakého správce souborů. Jinak po stisknutí tlačítka "Choose File" nebudete moci soubor na telefonu vybrat. Já na telefonu používám file manager "ES File Explorer" a jsem s ním spokojený. Podle různých informací na internetu je možné že některé telefony možná na toto tlačítko ani reagovat nebudou, ale moje testy na různých verzích Android 4+ byli úspěšně. Pokud nebudete moci naimportovat Excel z telefonu, tak mi napište nebo jej naimprotujte z počítače.

Zde je krátký návod pro import z telefonu.

Otevřete poslední záložku CREATE NEW v sekci My Lessons. Pak stisněte "Nahrát nové kartičky" a pak "Soubor mám na počítači".

Během editace lekce rozklikněte sekci
Přidat slovíčka ze souboru MS Excel
a klikněte na tlačítko Choose File a vyberte soubor s MS Excel šablonou. Zaškrtněte volbu Souhlasím s podmínkami používání aplikace a klikněte na tlačítko

Slovíčka se ihned nahrají a můžete je v případě potřeby doeditovat ručně.

Import slovíček z počítače je momentálně udělaný příliš složitě a plánuji to předělat na autentikaci pomocí emailové adresy a automatické stažení do telefonu. Na telefonu si v MyLessons kliněte na záložku CREATE NEW, pak "Nahrát nové kartičky" a pak "Soubor mám na počítači". Zobrazí se instrukce a otevřete si odkaz na počítači. Po uložení lekce se nově vytvořená lekce objeví v druhé záložce "PUBLIC WEB" nebo "PRIVATE WEB".
